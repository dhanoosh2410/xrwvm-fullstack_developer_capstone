# fullstack_developer_capstone

## Repository Name
xrwvm-fullstack_developer_capstone

## Project Overview

This project is a full stack dealership web application built as part of the Full Stack Application Developer capstone. The application provides a starter Django-based backend and frontend structure that can be extended with additional pages and features.

In this project, the main objective is to run the Django application locally, configure static files correctly, and add new static pages such as **About Us** and **Contact Us**. The project helps demonstrate basic full stack development skills such as project setup, configuration, routing, template rendering, and version control with GitHub.

## Objectives

The main goals of this capstone are:

- Set up and run the Django application in the lab environment
- Configure the project to serve static frontend files
- Create and update static HTML pages
- Add URL routes for the new pages
- Test the application in the browser
- Push completed changes to a GitHub repository for submission

## Features Completed

This project includes the following work:

- Django application setup and execution on a development server
- Static file configuration in `settings.py`
- Database migrations for the Django project
- Homepage rendering through the provided frontend
- Added **About Us** page
- Added **Contact Us** page
- Configured navigation links for static pages
- Updated Django URL patterns for page routing
- GitHub integration for version control and submission
